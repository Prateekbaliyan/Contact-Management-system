# Contact Management App

## How to Run
1.  **Install Maven**: Ensure Apache Maven is installed and added to your PATH.
    -   Download: [https://maven.apache.org/download.cgi](https://maven.apache.org/download.cgi)
    -   Guide: [https://maven.apache.org/install.html](https://maven.apache.org/install.html)
2.  **Run the App**: Double-click `run.bat`.

## API Endpoints
-   **Base URL**: `http://localhost:8082`
-   **Contacts**: `/contacts` (GET, POST)
-   **Contact Detail**: `/contacts/{id}` (PUT, DELETE)
